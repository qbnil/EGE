      • Формулировки
        • максимальное / минимальное
          • Для начальных значений можно использовать бесконечности: float('inf')
        • делится / не делится / даёт нужный остаток / заканчивается на какую-то цифру
          • a[i] % x == 0
          • a[i] % x != 0
          • abs(a[i]) % x == y
          • abs(a[i]) % 10 == x
          • Помним про остатки отрицательных чисел
        • N-значное
          • len(str(a[i])) == 3
          • 100 <= a[i] <= 999
          • a[i] in range(100, 1000)
          • Помним про отрицательные числа
        • положительное / отрицательное
        • больше / меньше другого элемента пары
          • Контролируем (не)строгость знаков
        • оба элемента / хотя бы один элемент / ровно один элемент пары
          • "ровно один" - исключающее "или": !=, ^
          • Помним про приоритет операций
        • не менее / не более N элементов в тройке
          • Предпочитаем счётчик длинному условию

    • 26. Обработка чисел с использованием сортировки
      • Эффекты сортировки
        • Известно, где находится много минимальных/максимальных значений
        • Одинаковые и близкие значения оказываются рядом

        • Читать и понимать сюжет задания, пользоваться примером для уточнения понимания
        • Открывать текстовый файл в Excel/Calc (по 1 числу в ячейке)
        • Сортировать, правильно определяя направления и ключи сортировки и потом правильно учитывая направление
      • По возможности предпочитаем решение в табличном процессоре, оно нагляднее

                =B1+A2

            • Решение программой

              with open('input.txt') as f:

                  # Считываем первую строку (предел суммы и кол-во файлов)

                  m, n = map(int, f.readline().split())

                  a = [int(x) for x in f]

              # Сортируем файлы по неубыванию

              a.sort()

              # Выбираем файлы по порядку, накапливая сумму

              k = 0

              s = 0

              while s <= m:

                  s += a[k]

                  k += 1

              # Отходим на один файл назад, т. к. превысили предел суммы

              k -= 1

              s -= a[k]

              # Определям запас места на диске и макс. размер реального файла

              d = m - (s - a[k])

              t = max([x for x in a if x <= d])

              print(k, t)
